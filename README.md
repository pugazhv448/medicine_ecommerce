# Medicine Drone Delivery E-Commerce

A modern, responsive e-commerce web application simulating an autonomous drone delivery service for medicines. Built entirely with frontend technologies (HTML, CSS, JavaScript) using `localStorage` for data persistence.

## Features
- **Drone Tracking:** Simulated real-time tracking of drone delivery status with an interactive map.
- **Shopping Cart:** Add, remove, and modify quantities with persistent local storage.
- **Product Filtering:** Search and filter medicines by category.
- **Dark Mode:** Seamless theme toggling for accessibility.
- **Checkout Simulation:** End-to-end checkout flow with form validation and simulated payments.
